## How to Run Locally

If you want to run this project on your local machine, follow these steps:

**Prerequisites**: Ensure you have Node.js & npm installed.

```sh
# Step 1: Clone the repository
git clone [https://github.com/mujahidraj/digital-trust-showcase.git](https://github.com/mujahidraj/digital-trust-showcase.git)

# Step 2: Navigate to the project directory
cd digital-trust-showcase

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
